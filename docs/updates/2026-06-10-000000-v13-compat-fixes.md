# 2026-06-10 - Foundry v13 Compatibility Fixes

## Summary

- Prepared `0.3.2`.
- Audited every selector and hook against the installed Foundry v13 (build 351) client source.
- Fixed the two features that were silently broken on v13: player sidebar tab hiding and the chat sticker dock after sidebar collapse/expand.
- Stopped the theme shell from painting v13 HUD overlays as paper panels.

## Changes

- `scripts/dnd-animal-ui.js`
  - `getSidebarTabButtons` now also matches `#sidebar nav.tabs [data-tab]` (v13 has no `#sidebar-tabs` id).
  - Active-tab detection reads `aria-pressed`/`.active` buttons and `ui.sidebar.tabGroups.primary`; activation prefers `Sidebar#changeTab` over the deprecated `activateTab`.
  - Sticker dock rework: removed the `#chat-message` input wrapper entirely. Core v13 `ChatLog#_toggleNotifications` re-parents `#chat-message` and `#chat-controls` when the sidebar collapses/expands or chat pops out, which stranded the wrapper and orphaned the button. The dock is now appended to `#chat-controls` (falls back to the chat form on v11/12), so core moves it together with the native controls. Added a `renderChatInput` hook as a re-injection safety net. `removeStickerButton` still cleans up the legacy `<=0.3.1` wrapper.
  - `sendSticker` uses `author` on v12+ (`user` is deprecated).
  - `DndAnimalThemeConfig#_updateObject` reads `this.form` instead of `event.currentTarget` (null on programmatic submits).
- `styles/dnd-animal-ui.css`
  - Hidden player tab/panel classes are now matched globally instead of requiring a `#sidebar-tabs` ancestor.
  - The core shell rule and its texture `::before` exclude `.faded-ui`, `.placeable-hud`, `#camera-views`, and `#tooltip` so v13 overlays (token HUD, hotbar, scene controls, players, camera dock) keep their native transparent look.
  - Player list: dropped the forced warm-brown text on `#players` lists (it sat on dark plates at roughly 2:1 contrast and erased per-user colors); only the v12-and-earlier `h3` heading gets a light color.
  - Added `#scene-navigation` (v13) theming: paper scene pills, brown scene names, blue active/viewed scene, themed expand toggle.
  - Sticker dock CSS rewritten as an inline 34px control inside `#chat-controls`; the panel anchors to the button (`right: 0; bottom: calc(100% + 10px)`). Removed the obsolete input-wrapper and 0.3.1 left-offset rules.
- `module.json`
  - Version `0.3.2`, `compatibility.minimum` raised to `11`, download URL bumped.
