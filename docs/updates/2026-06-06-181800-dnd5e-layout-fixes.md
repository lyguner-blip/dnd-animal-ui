# DnD5e Layout Fixes

Fixed the first visual pass after in-world screenshots showed over-broad theme selectors affecting core dnd5e sheets.

## Changes

- Removed the extra lower-left panel backing from `#players`.
- Reworked directory folder headers into softer cream/teal rows.
- Excluded `.dnd5e2` descendants from global button and form-control theme rules.
- Stopped click press effects inside dnd5e sheets.
- Removed custom dnd5e ability score, meter, and pip styling that could hide values or collapse item/monster sheets.
