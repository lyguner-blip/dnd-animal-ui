# Initial Release

Created `dnd-animal-ui` as a Foundry VTT module using the mature `ui-buttons-spell-glow` structure and authorized `animal-island-ui` visual language.

## Included

- Foundry `module.json` with GitHub install metadata.
- Theme CSS for Foundry shell, controls, sidebar, chat, forms, windows, and DnD5e sheets.
- Settings script and Handlebars configuration window.
- Bundled Animal Island assets and local font files.

## Validation

- `node --check scripts/dnd-animal-ui.js`
- `module.json` JSON parse
- CSS local `url(...)` resource existence scan
