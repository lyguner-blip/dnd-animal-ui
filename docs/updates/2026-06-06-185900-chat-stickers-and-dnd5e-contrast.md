# Chat Stickers and DnD5e Contrast

Added a local animal sticker picker to the chat input area and improved dark dnd5e list/panel contrast.

## Changes

- Added six SVG stickers: duck, bear, cat, frog, rabbit, and owl.
- Injected a sticker button into the Foundry chat form.
- Clicking a sticker sends a local image chat card.
- Adjusted dark dnd5e list text toward light cream and gold accents to reduce harsh brown-on-dark contrast.
